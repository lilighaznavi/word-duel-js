### word-duel-js

A two-player word guessing game built with JavaScript

#### How it works

- Both players roll dice at the start of each round.
- The player with the higher roll starts first.
- Players guess letters of a hidden word.
- A correct guess allows the player to keep playing.
- An incorrect guess ends the turn and passes it to the other player.
- The game continues until one player reaches 100 points.

#### Goal

- Be the first player to score 100 points and win the game.
